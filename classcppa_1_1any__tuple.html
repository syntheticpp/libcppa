Detailed Description

Describes a fixed-length copy-on-write tuple with elements of any type.

Member Function Documentation

const std::type_info * cppa::any_tuple::type_token ( ) const

Returns either &typeid(util::type_list<Ts...>), where Ts... are the element types, or &typeid(void).

The type token &typeid(void) indicates that this tuple is dynamically typed, i.e., the types where not available at compile time.
